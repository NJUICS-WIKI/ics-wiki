# 高等代数2

## 课程概述
作为高等代数选项的第二个学期，还是和AI的一起上课

## 课程内容
使用北大《高等代数》5-7章，重点是矩阵特征值、线性空间、欧式空间相关的内容

### 22级反馈
22级大部分人认为不是很难，但是由于sfr的个性课比较水，不是很有去的必要，基本靠自学和考前刷题

## 平时作业

教材上的题目，数量中等

特色是布置及其混乱，基本上没有人知道每一次作业提交应该提交到哪里，导致期末平时分基本靠缘分
